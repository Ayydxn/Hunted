# Hunted

---

Hunted is a plugin for Minecraft servers running [Paper](https://papermc.io) (or any of its parents such as [Spigot](https://www.spigotmc.org)) which adds the Minecraft Manhunt minigame. The plugin aims to recreate the same gameplay that can be seen in [Dream's Minecraft Manhunt series](https://www.youtube.com/playlist?list=PLChiKQUYPsJ9_ZXHeT287Ry2tIX1AiSkq) while adding some new features.

## 🛠️ Usage

### 📃 Requirements

- A JDK (Java Development Kit) for Java 21.
    - I recommend using the [Azul Zulu]() distribution as this is what the plugin is built with. However, any JDK distribution should work perfectly fine.

- Any Java IDE such as [IntelliJ IDEA](https://www.jetbrains.com/idea/) or [Eclipse](https://www.eclipse.org/ide/). You may also use other code editors with support for Java and Gradle such as [Visual Studio Code](https://code.visualstudio.com/).
    - If you're new to making plugins, I recommend using IntelliJ IDEA as it is what is used by most people for plugin development. It is also what is used in most plugin tutorials.

After cloning the project, you'll want to open it in your chosen code editor and wait for Gradle to load the project. Once the project has loaded, you can start coding!

To build the plugin, you can run the default `build` task, and it will be built. The built JAR can be found in the `build/libs` folder.
